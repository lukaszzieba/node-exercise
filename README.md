# Sport event parser

## Description

I am using node test runner, so if you want to run test's please use node v.21

## Install

It depends of your package manager. I use yarn but you can use npm.

```sh
yarn
or
npm install
```

## Run developmet

```sh
yarn dev
or
npm run dev
`

## Production build

```sh
yarn build
or
npm run build
`
## Run production build

```sh
yarn start
or
npm start
`


## Run linter

```sh
yarn lint
or
npm run lint
`

## Run tests

```sh
yarn test
or
npm test
```
